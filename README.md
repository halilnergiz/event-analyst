# Event Analyst
This project using for analyze the participants of events with getting event images.


build: 
```
npm install
```

run:
```
npm run start
```

Use the `event-analyst-fake-api` for app API 
``` 
https://github.com/halilnergiz/event-analyst-fake-api
```
